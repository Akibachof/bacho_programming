勉強日記 2021/08/15

目標

1. ページ上部にページ内容のリストを作り、クリックするとページ内ジャンプできるようにしたい
2. リスト表示された内容をタブ内に収納し、クリックすると表示されるようにしたい
3. 2.で作成したタブをクリックした時の動作を滑らかにしたい

1.1. リストの作成

http://webfeelfree.com/css-ul-ol-li-center/ と https://uxmilk.jp/60712 を参考にした。
html本文にulを設け、id="nav"とした。idはcrassとは異なり、同じindexファイルの中で一度しか使えないらしい。
そしてその内部にliを設け、各liの中にページ内のテーマ(Pythonとかその他とか)を作成した。
次にcssをいじる。cssに
#nav li { display: inline }
を加えてやることで、liの要素が横一列になる。

1.2. ページ内ジャンプ

1-1で作成したliでは、a href="#hoge" を挟んでいる。これをつけることで、li要素をクリックした時に 同じid名を持っている場所までジャンプできるようになる。すなわち、
id="hoge"
を持つ場所までページ内ジャンプができるようになる。
参考：https://seolaboratory.jp/42119/


2. リストの要素をタブ内に収納

CSSでアコーディオンをカスタマイズを参考にした。
タブ内に要素をしまうことを通称"アコーディオン"というらしい。書き方は以下の通り 
      
        <details>
            <summary>pandas</summary>
            <p><a href="URL">ショートカット名<p></a>
        <details/>
これにより、
pandas

pandas公式

乃木坂46公式サイト


のような表示が可能となる。また、上記のコードを(<details/>とか丸ごと)ページに直接表示するために、"実態参照文字"を使っている。


3. アコーディオンの動きを滑らかにする

2.の時と同じくCSSでアコーディオンをカスタマイズを参考にした。
結論から言えばcssをいじれば良い。 
        details[open] .News {
          animation: fadeIn 1.0s ease
        }

        @keyframes fadeIn {
          0% {
            opacity: 0; /* 透明 */
            transform: translateY(-10px); /* 上から表示 */
          }
          100% {
            opacity: 1;
            transform: none;
          }
        }
を入れてやることで所望の動作が得られる。
今回はアコーディオンの中に入る要素にclass="News"を入れているのでこのような形となっている。
